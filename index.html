Firebase Configuration

✅ Firebase is pre-configured! Your "ostend-is-music" project is ready for real-time sync.
You can update the configuration below if needed, or continue using the current setup.

How to get Firebase Configuration
  1. Go to Firebase Console
  2. Create a new project or select existing project
  3. Go to Project Settings (gear icon) → General tab
  4. Scroll down to "Your apps" section
  5. Add a web app or select existing web app
  6. Copy the configuration values from the Firebase SDK snippet
  7. Don't forget to enable Firestore Database and Authentication!

EmailJS Configuration

Configure EmailJS to send booking confirmation emails to users automatically.

How to get EmailJS Configuration
  1. Go to EmailJS.com and create a free account
  2. Create an email service (Gmail, Outlook, etc.)
  3. Create an email template with these variables: {{user_name}}, {{user_email}}, {{artist_name}}, {{table_name}}, {{speakers}}, {{event_date}}, {{time_slot}}
  4. Get your Service ID from the Email Services page
  5. Get your Template ID from the Email Templates page
  6. Get your Public Key from Account Settings → API Keys
